Boards + Auto Setting

Your home wall. Infinite fresh problems.

Turn any spray wall, home woody or systems board into a digital twin — then let BoulderLog set the routes for you.

Build the board once. Set real geometry — width, height, wall angle — mark the kickboard and height zones, and tag each hold's type and orientation. The board remembers forever.
Generate with one tap. Dial in target grade, length and style focus for a fresh, never-seen problem — linear or endurance circuit, every route grade-estimated as it's built.
Routes that actually climb. A biomechanical body model simulates both hands and your center of mass, checking reach at your height — so problems move like a human setter built them.
Whole sessions, auto-set. Generate an entire pyramid session — warmup → build → work → limit — as a stack of fresh routes in a single tap.
